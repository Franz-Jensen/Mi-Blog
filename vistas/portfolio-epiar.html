Sistema web para la gestión académica de los estudiantes de la E.P Ingeniería Agrónoma

Este sistema web de gestión académica esta desarrollado en PHP y MySQL, permite gestionar facilmente y tener el control adecuado de toda la información correspondiente a la carrera profesional de Ingeniería Agrónoma, esto corresponde a información como de los postulantes, estudiantes, gestión de notas, cursos, promedio ponderado del alumno, planes de estudio, pregrado, postgrado, reportes, convenios, entre otros. Manteniendo un control eficaz y permitiendo monitorizar la completa información de todos los datos institucionales.
